# Market Basket Analysis using Python

## Growfinix Machine Learning Internship

### Task 4: Retail - Market Basket Analysis (Association Rules)

---

## Objective

The objective of this project is to analyze customer purchase behavior and identify relationships between items frequently bought together in retail stores.

Example:
Customers who buy **Bread** are also likely to buy **Milk**.

---

## Concept Used

* Association Rule Mining
* Support Calculation
* Confidence Calculation
* Apriori Logic (Manual Implementation)

---

## Technologies Used

* Python
* Pandas
* itertools

---

## Project Workflow

1. Create sample transaction dataset
2. Extract unique items
3. Calculate support for itemsets
4. Identify frequent itemsets
5. Generate association rules
6. Display relationships between products

---

## Output Includes

* Transaction dataset
* Frequent itemsets with support values
* Association rules with confidence values

---

## Example Result

* Bread → Milk
* Milk → Diaper
* Diaper → Beer

These rules help in building **recommendation systems in e-commerce and retail stores**.
